# matplotlib-plotly-plots

Python plots for two mathematical functions using **Matplotlib** and **Plotly**. Compares static and interactive visualizations for a quadratic function and a damped cosine wave.

## Features

- 📊 **Quadratic Function** plots with both libraries
- 🌊 **Damped Cosine Wave** visualizations
- 🔄 Side-by-side comparison of Matplotlib vs Plotly
- 🎯 Interactive plots with Plotly
- 📈 Static high-quality plots with Matplotlib

## Installation

```bash
git clone https://github.com/ReetiShah/matplotlib-plotly-plots.git
cd matplotlib-plotly-plots
pip install -r requirements.txt
```

## Requirements

```
matplotlib
plotly
numpy
pandas
```

## Usage

Run the main script to generate all plots:

```bash
python main.py
```

Or run individual plot scripts:

```bash
python quadratic_plot.py
python damped_cosine_plot.py
```

## Matplotlib vs Plotly

| Feature | Matplotlib | Plotly |
|---------|-----------|--------|
| Interactivity | Static | Interactive |
| Learning Curve | Steep | Moderate |
| Customization | High | High |
| Best For | Static reports | Web dashboards |
